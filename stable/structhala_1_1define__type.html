Detailed Description

template<class VectorLike>
struct hala::define_type< VectorLike >

Template that defined the type of the vector variable.

Given the VectorLike class, the structure must define a value_type member type, i.e., using value_type = scalar_type, where scalar_type is the type of the vector entries (not including qualifiers). The included implementation will automatically handle any VectorLike class which defines a member typedef value_type in the same way as the std::vector class. Must be specialized to handle VectorLike classes that do not provide value_type.
