Detailed Description

template<typename T, class VectorLike>
struct hala::binded_gpu_vector< T, VectorLike >

A struct that wraps around a gpu_vector and a reference to a cpu vector-like.

At the constructor stage, the data from the cpu-vector will be loaded into the gpu vector, when the destructor is called, the data currently set in the gpu vector will be written back. This allows for RAII style of management of temporary data that needs to be placed back on the CPU after the call.

The struct can be used as a gpu vector like class.
