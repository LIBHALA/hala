Detailed Description

template<class VectorLike>
struct hala::define_data< VectorLike >

Struct with a single function that returns a raw pointer to the internal contiguous data stored in the vector, uses .data() method.

Unlike the get_data() function, the data_extractor class can be partially specialized, so a single template can be used to cover many cases, e.g., std::valarray<float> and std::valarray<double>.
