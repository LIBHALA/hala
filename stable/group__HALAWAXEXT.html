Detailed Description

Defines common methods that require interoperability between multiple HALA modules.

  • hala::mixed_engine allows for all BLAS and Sparse methods to be used with input vectors on the CPU while the operations are performed on the GPU (data is moved behind the scenes)
  • BLAS-like extensions that utilize one or more of the existing BLAS methods (and sometimes simple additional algorithms) that perform extra functionality, e.g., batch dot and axpy operations
