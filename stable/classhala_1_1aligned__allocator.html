Detailed Description

template<class T, regtype R = default_regtype>
class hala::aligned_allocator< T, R >

Allocator that aligns memory to the selected register.

Very simple allocator that uses the C11 aligned_alloc to make an aligned memory segment. The method part of the C (not C++) 2011 standard and required the inclusion of stdlib.h header.
