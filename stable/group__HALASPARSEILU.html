Detailed Description

Preconditioner
The incomplete lower-upper factorization is a method to approximate a sparse matrix by the produce of two sparse triangular matrices with patterns very similar to the original. The ILU method is a proven general preconditioner for Krylov iterative solvers, such as those included in the HALA Solver Algorithms. HALA provides a reference CPU implementation of the algorithm and wrappers around the ILU methods implemented in the cuSparse library.

Constructs an ILU preconditioner associated with a compute engine.

Constructs an ILU factorization of a sparse matrix using the given policy.

  • cengine is the compute engine associated with the matrix and factor
  • pntr, indx and vals describe a sparse matrix in row-compressed format
  • policy, if set to lower/upper case L, indicates whether to use the level-policy and algorithms when computing the hala::cuda_ilu factor; the variable has no effect when using the cpu_engine.

Overloads:

make_ilu(engine, pntr, indx, vals, policy);

An overload is provided for each compute engine, the intended use of this factory method is in conjunction with the auto keyword:

auto ilu_factor = make_ilu(engine, pntr, indx, vals, policy);

Thus the cpu/cuda/mixed variant of the factor will be inferred from the type of the engine.
