Detailed Description

Solvers
Covers the general factorization \( A = PLU \) and the symmetric positive definite Cholesky \( A = L L^T \). Both the factorization and the solve stages are covered here.

Wrapper to LAPACK Cholesky decomposition of an s.p.d. matrix xpotrf()

Computes \( A = U^T U \), or \( A = L L^T \), A is overwritten with U/L

  • uplo must lower or upper case U or L, indicates whether to work with the upper (U) or lower (L) part of A
  • A must be a symmetric positive definite matrix of size lda by N
  • lda is at least N

Overloads:

hala::portf(uplo, N, A, lda);
hala::portf(uplo, N, A, lda = -1);
hala::portf(engine, uplo, N, A, lda);
hala::portf(engine, uplo, N, A, lda = -1);

All overloads accept an optional last argument a workspace vector that will be resized to hala::portf_size(). In addition, lda of -1 will be overwritten with N.

Wrapper to LAPACK solver using Cholesky decomposition of an s.p.d. matrix xpotrs()

Solves the linear system of equations \( A X = B \), where B is overwritten with the solution.

  • uplo, N, A, and lda must already come from a call to hala::potrf().
  • B is an nrhs by ldb matrix
  • ldb is at least N
  • B is overwritten with the solution

Overloads:

hala::potrs(uplo, N, nrhs, A, lda, B, ldb);
hala::potrs(uplo, N, nrhs, A, B, lda = -1, ldb = -1);
hala::potrs(engine, uplo, N, nrhs, A, lda, B, ldb);
hala::potrs(engine, uplo, N, nrhs, A, B, lda = -1, ldb = -1);

Note, lda or ldb of -1 will be overwritten with N.

Wrapper to LAPACK general PLU factorization xgetrf()

Computes \( P A = L U \), A is overwritten with U/L and the permutation matrix P is stored in ipiv

  • A must be M by N matrix with full column/row rank (depending on the smaller of M/N)
  • A has size lda by N with lda begin minimum M
  • ipiv is a vector of integers holding the permutations, will be resized to match min M, N

Overloads:

hala::gertf(M, N, A, lda, ipiv);
hala::gertf(M, N, A, ipiv, lda = -1);
hala::gertf(engine, M, N, A, lda, ipiv);
hala::gertf(engine, M, N, A, ipiv, lda = -1);

All overloads accept an optional last argument a workspace vector that will be resized to hala::gertf_size(). In addition, lda of -1 will be overwritten with M.

Solves the linear system of equations \( A X = B \), where B is overwritten with the solution.

  • N, A, lda, and ipiv must already come from a call to hala::getrf() with M = N
